# Gemini CLI Directory Map

At the top level of the `gemini-cli` submodule:

- `packages/cli/` - Ink UI, CLI commands, non-interactive mode, sandbox bootstrap.
- `packages/core/` - Core runtime: prompts, routing, tools, policy, telemetry.
- `packages/a2a-server/` - HTTP agent interface for multi-client access.
- `packages/vscode-ide-companion/` - IDE companion integration.
- `docs/` - user and developer documentation.
