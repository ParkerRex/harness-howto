# Cline Comparison Notes

Highlights when comparing Cline to other agents in this repo.

- Surface: VS Code extension with a webview UI and extension-host runtime.
- Tool safety: explicit approvals for tool execution.
- Persistence: per-task storage under `~/.cline/data/tasks/<taskId>` plus global
  settings.

See `../../comparison/summary.md` for a cross-project snapshot.
