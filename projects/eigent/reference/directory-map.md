# Eigent Directory Map

At the top level of the `eigent` submodule:

- `src/` - renderer UI (React + Zustand), SSE client, task logs.
- `electron/` - Electron main process, backend spawn, MCP config.
- `backend/` - FastAPI AI runtime, CAMEL workforce, tools.
- `server/` - optional persistence and sharing backend (FastAPI + DB).
- `docs/` - conceptual docs and notes.
